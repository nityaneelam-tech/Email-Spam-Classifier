# 📧 Email Spam Classifier

This is a simple **Python project** that detects whether an email message is spam or not using **Machine Learning (Naive Bayes)**.

---

## 🎯 Features
- Uses **Naive Bayes algorithm**  
- Converts text into numbers using **TF-IDF Vectorizer**  
- Predicts whether an email is **SPAM or NOT SPAM**  
- Easy to run in **Command Prompt**

---

## ⚙️ Installation

1️⃣ Install Python (if not already installed)  
2️⃣ Install the required library:
```bash
pip install scikit-learn
```

---

## ▶️ How to Run

```bash
python email_spam_classifier.py
```
